# 🎬 Movie Showcase Portfolio
A custom-built interactive cinema website featuring a dynamic carousel, smooth section switching, and protected video content.

## ✨ Features
* **Custom Carousel:** Interactive slider for browsing featured movies.
* **Smart Navigation:** Single-page application (SPA) style switching between Home, Movies, and Contacts.
* **Video Protection:** Anti-download measures and autoplay-ready configurations.
* **Responsive Design:** Works on desktop and mobile.

## 🛠️ Built With
* HTML5
* CSS3 (Flexbox & Grid)
* Vanilla JavaScript (ES6)
